## Anti-pattern reminders

- **Don't tight-poll.** When halted (waiting on builder/CI/Owner),
  `sleep 600s` between re-checks. After 6 idle cycles (~1h) without
  progress, exit autonomous goal — Owner re-triggers when ready.
- **Don't misdiagnose.** If a bundle's bug repros differently than the
  issue describes, STOP and ping Advisor.
- **Don't invent tools that aren't exposed.** If a 3rd-party connector
  is needed, halt — Owner-action, not CEO loop.
- **Don't conflate G outcomes.** Bundles β/γ/δ/ζ proceed regardless of
  G. Only ε is conditional. Don't gate the wrong bundles on G.
- **Don't skip the release dance for δ.** After aiplus PR merges, you
  must cut v0.5.28 (or comment on AEL milestone requesting advisor cut
  it), THEN bump AEL vendor, THEN merge AEL PR, THEN cut ael v0.1.4.
  Going out of order means AEL prompt update lands without the
  substrate it depends on.
